Definition of the levels

  1. None: I don't know anything about this
  2. Novice: I understand what it's doing but I'm not able to do it on my own.
  3. Junior: I've got the baseline of this profession and can create easy things
  4. Medior: I've got solid knowledge of this profession and know how to achieve things with it
  5. Senior: Others ask me for help in this topic and basically I know everything about this. I understand problems so good that I'm able to explain this
  6. Master: total nerd, I'm so good in this topic that I can question the profession itself.
